# 生産活動報告システム

## 画面

- 利用者フォーム: `index.html`
- 管理者画面: `admin.html`

## 管理者パスワード

現在テスト版につき、初期状態では未記入でログインできます。管理者画面の「項目編集」から変更できます。

## 保存方式

データはブラウザの `localStorage` に保存されます。同じ端末・同じブラウザでフォーム入力と管理画面の集計を確認できます。

## 主な機能

- 氏名、活動内容、所要時間、進捗状況の報告
- 生産活動項目の追加・編集・非表示
- 週単位、月単位の集計
- 全体グラフ、個人別グラフ、活動別統計、個人別統計
- 報告一覧とCSV出力
